```text
Digital pathology and artificial intelligence
数字病理学与人工智能
诊断病理学作为癌症诊断的基石，正逐步向数字化工作流程转变。数字技术作为基于计算机的产品和解决方案的统称，已在病理学领域应用多年。然而，近期引入的数字全玻片成像技术已经改变了该领域的许多方面并引发了革命。“数字病理学”一词常用于涵盖全玻片成像技术以及基于机器学习/人工智能（AI）的方法。作为病理信息学的一个分支，数字病理学将虚拟显微镜与基于计算机的工具相结合。数字病理学的核心原则详述如下。
全玻片成像
高分辨率扫描仪将玻璃切片转化为高分辨率数字图像，从而实现无缝共享、高级分析功能以及更便捷的归档。全数字化诊断工作流程于2015年首次在少数实验室实施，此后在全球范围内稳步发展并不断扩展。业已证明，数字化诊断是安全的，且非劣效于显微镜诊断。数字化工作流程具有诸多优势，尤其在具备数字档案库时更为显著。这些优势包括：能够远程工作、更短的周转时间、准确的标注与快照、免去为准备会议而搜寻切片的过程、更快捷地复核旧病例以及更易于与同行会诊、不受地点限制的多观察者共同阅片、改善组织的概览与追踪以确保无组织碎片遗漏、图像与实验室信息系统自动关联以防止标本混淆、与放射系统的整合，以及强化针对学生的数字化教学。
图像分析
图像分析运用综合的图像分析手段与基于AI的算法，对数字组织图像中的异常进行检测、量化并分类。基于AI的方法的一大关键优势在于其可扩展性，使其能够在相对较短的时间内处理海量数据——在这一点上可能超越人类的能力。在乳腺癌的背景下，数字病理学的临床获益涵盖了广泛的应用领域，从癌症检测、亚型分类与组织学分级，到预测治疗反应或疾病进展轨迹（例如，患者生存概率），再到捕获分子改变，例如检测体细胞驱动突变。
AI的一个关键组成部分是机器学习，其中包括强化学习、无监督学习与监督学习。深度学习作为基于卷积神经网络的AI子集，在处理复杂数据方面于数字病理学中尤为有效。目前已开发出数个在海量数据集上训练的大规模AI基础模型，使它们只需极少的微调即可执行多项任务。这些模型通常基于深度学习架构，利用视觉Transformer模型（vision transformer model），支持在自然语言处理与多模态学习中的应用。许多实用的AI算法已经超越了初始的科研阶段，目前已实现商业化。当前，乳腺病理AI应用涵盖了乳腺病理学家日常工作中遇到的广泛任务，包括：生物标志物染色质量控制、浸润性癌和导管原位癌的检测、微小钙化的识别、淋巴结转移的检测、旨在提高分级可重复性的核分裂象计数，以及对Ki-67阳性、ER阳性和PR阳性细胞比例的客观评估，还有ERBB2（HER2）和PDL1评分。然而，目前尚无任何单一工具能够涵盖乳腺病理学报告的所有要素。
图像分类近期也已扩展至更为复杂的任务中，通常被称为高级计算病理学。AI驱动的方法再次印证了基因型与表型之间已知的关联 ，并有望得到进一步发展以揭示新的预后特征。研究表明，AI能够判定驱动基因突变的分子特征、诸如肿瘤突变负荷之类的基因组改变，以及同源重组缺陷。此外，AI正被应用于更专业的任务，例如乳腺癌的亚型分类与预测治疗反应。其中许多已获得了某种形式的认证，从CE体外诊断（CE-IVD）标志到体外诊断医疗器械法规（IVDR）以及美国食品药品监督管理局（FDA）的批准不等。预计在不久的将来，还会推出更多AI算法（例如用于肿瘤浸润淋巴细胞和淋巴血管侵犯的算法）。基于AI的工具的采用同样在改进工作流程方面具有巨大潜力。将AI的能力与其他与患者相关的影像学、临床以及组学数据相结合，可创造出一种增强患者护理水平的强大工具。然而，实施过程仍面临挑战，包括需要解决或完善商业模式，因为在大多数国家，使用AI将无法获得直接的费用报销。其他挑战还包括科学与监管障碍、责任归属、信任度、经济影响以及环境影响。
```
